# Translation Tool - Multi-language translation utility

**Package**: `@apps/translate`
**Version**: 0.1.0
**Repository**: `pkg-apps-translate`

## Overview

Translation Tool - Multi-language translation utility

## Installation

### Prerequisites

This package requires the following external commands:

- `dmenu`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @apps/translate

# Or install from local source
cd ~/.pkgs
stow translate
```

## Usage

### Application

Translation Tool - Multi-language translation utility

Run the application to see usage instructions.

## Configuration

Configuration files are typically stored in:
- `~/.config/translate/` - User configuration
- `~/.local/share/translate/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @apps/translate

# Or manual unstow
cd ~/.pkgs
stow -D translate
```
